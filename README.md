**IP Subnet Hesaplayıcı (IPv4)**  
Basit, interaktif ve güçlü bir Python aracı ile verdiğiniz bir IPv4 adresini istediğiniz sayıda alt ağa (subnet) bölebilir, tüm detayları (ağ adresi, yayın adresi, kullanılabilir IP aralığı, maskeler ve ikilik gösterim) ile birlikte görüntüleyebilirsiniz.

---

## Özellikler

- IPv4 CIDR desteği  
- Otomatik subnet hesaplama (2ⁿ tabanlı)  
- Her subnet için:
  - Ağ Adresi
  - Subnet Mask
  - İlk & Son Kullanılabilir IP
  - Broadcast Adresi
  - Kullanılabilir Host Sayısı  
- Binary gösterim desteği  
- Çıktıyı `.txt` dosyasına kaydetme  
- Hatalı girişler için yönlendirme yapan sağlam doğrulama  
- Windows, Linux, macOS uyumlu
